EcoVision Bot — Telegram-бот, помогающий снижать вклад человека в глобальное потепление
через правильную утилизацию отходов.

Основная идея:

• Определение типа мусора по фотографии
• Подсказка по правильной утилизации
• Формирование экологически ответственного поведения

Проблема:

• Рост свалок и сжигания отходов
• Увеличение выбросов парниковых газов
• Неправильная сортировка мусора

Решение:

• Анализ изображения отхода
• Краткая инструкция по утилизации
• Пояснение экологической пользы

Цель проекта:

• Снижение выбросов CO₂
• Повышение экологической грамотности
• Вклад в борьбу с глобальным потеплением

Также Наш бот распознаёт фото с помощью Google Techable Machine, проанализировавшию по 300 раз каждую из 9 групп по 300 изображений!

Итог:

• Простые ежедневные действия помогают сохранять климат и будущее планеты.
